= setModSetting

include::../_requiresAuthentication.adoc[]

Changes Fiscit Remote Monitoring Mod General Settings

== URL

`GET http://localhost:8080/setModSetting`

include::../_requestBody.adoc[]

| <Mod Session Setting Name> | <Data Type for Setting>

Current Supported Settings
| SplineSampleDistance | Float

|===

include::../_responseBody.adoc[]

| IsSent | Boolean | If the message passed all checks and was sent.
|===

== Example Request

[source,bash]
----
curl -X POST 'http://localhost:8080/setModSetting' \
-H 'Content-Type: application/json' \
-H 'X-FRM-Authorization: <your access token>' \
--data-raw '{ \
  "SplineSampleDistance": 55 \
}'
----

== Example Response

[source,json]
----
[
  {
    "IsSent": true
  }
]
----
