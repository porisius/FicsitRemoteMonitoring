= Get Crate Inv

This is inventory of the world in Dismantle and Death Crates.

== Similar

[cols="1,1,2"]
|===
|Resource|Endpoint|Description
| Inventory | xref:json/Read/getCloudInv.adoc[getCloudInv] | Gets a list of the cloud inventory.
| Inventory | xref:json/Read/getStorageInv.adoc[getStorageInv] | Gets a list of the cloud inventory.
| Inventory | xref:json/Read/getWorldInv.adoc[getWorldInv] | Gets a list of the world inventory.
|===

== URL

`GET http://localhost:8080/getCrateInv`

include::../_responseBody.adoc[]

| ID | String | Unique ID of the Crate.
| Name | String | Name of the Crate.
| ClassName | String | Class Name of the Crate.

{set:key-name:location}
{set:key-description:Location details of the Crate.}
include::../Models/_location.adoc[]

// Crate Type
| Type | String | Type of Crate (Dismantle / Death).

// inventory items
| Inventory | Object[] | List of items in the Crate.
include::../Models/_inventoryItem.adoc[]

// bounding box
{set:key-name:BoundingBox}
include::../Models/_boundingBox.adoc[]

// features
include::../Models/_features.adoc[]
|===
