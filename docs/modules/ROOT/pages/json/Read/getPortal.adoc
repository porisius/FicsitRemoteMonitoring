= Get Portal

Gets a list of all portals.

== URL

`GET http://localhost:8080/getPortal`

include::../_responseBody.adoc[]

| ID | String | Unique ID of the Portal.
| Name | String | Name of the Portal.
| ClassName | String | ClassName of the Portal.
